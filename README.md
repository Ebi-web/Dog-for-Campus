# Dog-for-Campus
大学からのお知らせを見落としてた！大学関連のホームページがいくつもあって見に行くの面倒！
CAMPUS-DOGに任せて時間とストレスを節約しましょう。監視するサイトを登録しておくだけで，更新された情報を
まとめておいてくれます。

対応予定の大学：東北大学(理学部)
